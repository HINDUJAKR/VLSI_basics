A collection of fundamental VLSI digital design concepts implemented in Verilog HDL and verified using Xilinx Vivado.
# VLSI Basics

This repository contains the VLSI concepts I refreshed during my internship at NIT Trichy.

## Topics Covered
- Basic Logic Gates
- Multiplexers
- Demultiplexers
- Decoders
- Adders
- Flip-Flops
- Counters
- Registers
- ALUs

## Tools Used
- Xilinx Vivado
- Verilog HDL
- Git & GitHub

## Repository Structure
- Verilog source codes
- Testbenches
- Simulated report

This repository serves as my learning record and reference for Digital Design and VLSI concepts.
